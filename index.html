    2. Knee Displacement Detection using IOT

      3. A knee displacement detector that uses an ESP32 and a time-of-flight sensor to detect the angle of the knee. With this project, I am able to display the live angle of the knee on an IoT app on a smartphone. This project is a great example of using technology to help monitor and manage health. I am proud of the work that I have put into this project and look forward to finding new ways to use technology to improve healthcare outcomes.

    3. Automated Saline Controlling System Using IOT

      4. The Saline Controlling System is an IoT-based system that allows for remote control and monitoring of saline flow in a hospital. The system uses an ESP32 module, and servo motor to regulate the flow of saline. A mobile app installed on the nurse's phone provides control of the system and alerts the nurse when the saline bottle is empty. The app also provides real-time status updates for each patient in the hospital, allowing for better management of patient care. A research paper has also been published for the same.

    4. Social Distancing With Python And ML

      5. Since we are going through a pandemic, everyones finance was getting affected in one way or the other and the majority of the problem lied where an industry / company was not able to run due to Covid19 reasons.
